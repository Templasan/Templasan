Projetos

Aplicação web de consulta e análise dos dados públicos de exportação/importações dos municípios de São Paulo.

Link para o projeto rodando Link para o repositório

Durante a primeira sprint de um projeto acadêmico baseado na metodologia Scrum, colaborei com uma equipe multidisciplinar no desenvolvimento de uma plataforma para visualização de dados de exportações do estado de São Paulo para o mercado internacional.

Minhas principais contribuições incluíram a organização e atualização dos backlogs do produto e da sprint, com foco na definição de requisitos funcionais e não funcionais. Também fui responsável por manter o repositório do GitHub, assegurando a documentação adequada e o controle de versão contínuo.

Desenvolvi o protótipo inicial da aplicação no Figma, priorizando uma interface amigável e funcional. Além disso, contribuí diretamente na elaboração de uma ferramenta de busca de dados, com o uso de Pandas para tratamento das informações e Google Colab para testes e visualizações interativas.

As atividades também abrangeram estudo e aplicação de SQL para consultas, limpeza de dados, expansão das buscas e representações gráficas, além da realização de pesquisas complementares envolvendo análise de sazonalidade, competitividade e comparativos entre diferentes períodos econômicos.
